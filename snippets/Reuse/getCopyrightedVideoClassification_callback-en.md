### 回调结果

错误码，具体请参见 [常见错误码\|_blank](!Common_error_codes)，如查询不到可联系 ZEGO 技术支持。

| 参数 | 类型 | 描述 |
| --- | ---- | --- |
| country | String Array | 国家或地区列表。 |
| genre | String Array | 视频题材列表。|
| release_year | Int Array | 视频发行年份列表。 |
