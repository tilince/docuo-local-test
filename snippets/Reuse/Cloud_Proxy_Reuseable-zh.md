## 功能简介

通过设置 SDK 的云代理接口，将 SDK 对应的所有流量通过云端的代理服务器中转，实现与 RTC、L3（[超低延迟直播\|_blank](!Publisher_Player_Advanced/Low_Latency_Live)）通信。

<div class="mk-warning">

- “云代理”功能是`付费`功能，如果您需要使用此功能，请联系 ZEGO 商务人员咨询。
- 开通云代理服务后，如果企业有配置防火墙，还需要将云代理使用的 IP、域名、端口加入企业防火墙白名单。
</div>

<img src="/Pics/Express/Cloud_Proxy.png" width=80%>
