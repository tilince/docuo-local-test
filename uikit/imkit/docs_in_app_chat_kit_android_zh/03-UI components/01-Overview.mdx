# 概述

UI 组件是 IMKit SDK 的基本单元。根据业务需求，您可以将所需的组件嵌入到相应的UI页面中。组件从 IMKit SDK 的实例接收数据，将数据发送到 ZEGO 服务器，并维护其业务逻辑。
