### generateKitTokenForTest

**函数原型：**

```ts
static generateKitTokenForTest(
    appID: number,
    serverSecret: string,
    roomID: string,
    userID: string,
    userName?: string,
    ExpirationSeconds?: number
): string;
```

**接口说明：**

根据 appID 和 serverSecret 生成用于测试的 kit token。

### generateKitTokenForProduction

**接口说明：**

将 [服务器生成的 token\|_blank](!KitToken_Authentication) 转换为 UIKit 需要用的 kit Token。

<div class="mk-warning">

为规避安全风险，在您的 App 正式上线时，请使用此接口生成 kit token，不要使用 `generateKitTokenForTest`。
</div>

**函数原型：**

```ts
static generateKitTokenForProduction(
    appID: number,
    token: string,
    roomID: string,
    userID: string,
    userName?: string
): string;
```
