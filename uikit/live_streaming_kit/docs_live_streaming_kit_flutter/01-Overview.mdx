## Change log

<Accordion title="Click here to know more" defaultOpen="false">
#### Version 2.2.0
Release date: 2023.02.28

**New features**:
1. Added `screenSharingController` in `ZegoUIKitPrebuiltLiveStreaming`, use this to customize the button that is used to invoke the full-screen mode interface.

#### Version 2.1.0
Release date: 2023.02.21

**New features**:
1. [PK battles](./04-Enhance%20the%20livestream/01-PK%20battles.mdx). 
2. Added `onLiveStreamingStateUpdate` in `ZegoUIKitPrebuiltLiveStreamingConfig`, use this to monitor the state of the live stream.

#### Version 2.0.0
Release date: 2023.02.07

**API changes**:

- Modified:

    Before: `Stream<List<ZegoSignalingInRoomTextMessage>> getInRoomTextMessageStream()`

    After: `Stream<ZegoSignalingPluginInRoomTextMessageReceivedEvent> getInRoomTextMessageReceivedEventStream()`

> You can still get the `List<ZegoSignalingInRoomTextMessage>` parameter from the new interface's class member `ZegoSignalingPluginInRoomTextMessageReceivedEvent.messages`.
