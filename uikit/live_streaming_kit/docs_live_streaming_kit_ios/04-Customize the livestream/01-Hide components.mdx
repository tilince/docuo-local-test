# Hide components

By default, the Live Streaming Kit (ZegoUIkitPrebuiltLiveStreaming) displays the user's avatar and sound waves when the camera is turned off.
If you are not satisfied with the user avatars and sound wave style, you can hide them using the `showSoundWavesInAudioMode` configuration.

- `showSoundWavesInAudioMode`: In audio mode, whether to display the sound waves around the user avatar in audio mode. Displayed by default.
