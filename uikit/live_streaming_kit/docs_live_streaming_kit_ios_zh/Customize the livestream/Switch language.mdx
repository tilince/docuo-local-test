# 切换语言


## 功能概述

ZEGO 各场景 UIKit 默认以英文展示，但开发者可以通过简单的代码参数设置，轻松将界面语言切换为中文。

## 实现流程

1. 通过设置 `ZegoUIKitPrebuiltLiveStreamingConfig` 类的 `translationText` 属性为 `ZegoTranslationText(language: .CHS)`，开发者可以将互动直播 UIKit 的界面语言切换为中文。

2. 在初始化 `ZegoUIKitPrebuiltLiveStreamingVC` 类型的变量时传入上述 `config`，即可切换语言为中文。

```swift
// 假设用户身份为房主，创建 ZegoUIKitPrebuiltLiveStreamingConfig 对象
let config: ZegoUIKitPrebuiltLiveStreamingConfig = ZegoUIKitPrebuiltLiveStreamingConfig.host()

// 将界面语言设置为中文
config.translationText = ZegoTranslationText(language: .CHS)

// 初始化 ZegoUIKitPrebuiltLiveStreamingVC
let liveVC: ZegoUIKitPrebuiltLiveStreamingVC = ZegoUIKitPrebuiltLiveStreamingVC(
    appID: yourAppID, 
    appSign: yourAppSign, 
    userID: yourUserID, 
    userName: yourUserName, 
    liveID: yourLiveID, 
    config: config)
```
