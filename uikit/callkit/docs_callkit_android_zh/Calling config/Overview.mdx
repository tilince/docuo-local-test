# 功能概述

---

**音视频通话 UIKit (Call Kit)** 提供了一组默认的行为和样式，同时我们支持根据您的需求，灵活地的定制或添加您的自定义业务逻辑。

在 [快速开始](../Quick%20start.mdx) 后，您可能会注意到在集成 Call Kit 时有一个 `ZegoUIKitPrebuiltCallConfig`，您可将其用于自定义配置。

## 功能列表

<CardGroup cols={2}>
<Card title="添加自定义组件" href="./Add%20custom%20components.mdx">
  在呼叫界面的顶层添加您的自定义组件。
</Card>
<Card title="调整布局" href="./Configure%20layouts.mdx">
  选择并配置您想要的布局，目前支持画中画和宫格式布局。
</Card>
<Card title="隐藏用户视图标签" href="./Hide%20the%20label%20on%20the%20user%20view.mdx">
  隐藏浮于用户视图上方的用户名、麦克风和摄像头状态图标。
</Card>
<Card title="实现纯语音互动" href="./Implement%20an%20audio-only%20call.mdx">
  设置摄像头和麦克风的初始状态，实现仅音频通话。
</Card>
<Card title="自定义菜单栏" href="./Customize%20the%20menu%20bar.mdx">

- 禁用菜单栏的自动隐藏功能。
</Card>
<Card title="实现挂断确认对话框。" href="./Set%20a%20hangup%20confirmation%20dialog.mdx">

</Card>
<Card title="配置呼叫邀请" href="./Call%20invitation%20config.mdx">

- 隐藏拒绝按钮。
- 修改 UI 文本。
- 基于监听回调实现更多业务逻辑。
</Card>
