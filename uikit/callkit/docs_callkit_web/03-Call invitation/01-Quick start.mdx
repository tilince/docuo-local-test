## Prerequisites

- Get the **AppID** and **ServerSecret** of the project.



## Integrate the SDK

### Initialize the Call Kit with call invitation feature

Initialize the Call Kit, and add the call invitation plug-in:
- Generate a [Kit Token](../04-Authentication%20and%20Kit%20Token.mdx)
- Fill in the required parameters in the following code.

```javascript
// Generate a Token by calling a method.
// @param 1: appID
// @param 2: serverSecret
// @param 3: Room ID
// @param 4: User ID
// @param 5: Username
const userID = ""; 
const userName = "userName" + userID;
const appID = 0;
const serverSecret = "";
const TOKEN = ZegoUIKitPrebuilt.generateKitTokenForTest(appID, serverSecret,null, userID, userName);

const zp = ZegoUIKitPrebuilt.create(TOKEN);
zp.addPlugins({ ZIM });
```


### Set a button-trigger event

Set up a button and add a button-click event for making call invitations.

You can customize the position of the button you set for making call invitations accordingly, passing in the ID of the user you want to call.


```html
  <button onclick="invite()">invite</button>
```

```javascript
function invite() {
   const targetUser = {
        userID:'',
        userName:''
    };
   zp.sendCallInvitation({
    callees: [targetUser],
    callType: ZegoUIKitPrebuilt.InvitationTypeVideoCall,
    timeout: 60, // Timeout duration (second). 60s by default, range from [1-600s].
   }).then((res) => {
    console.warn(res);
   })
   .catch((err) => {
   console.warn(err);
   });
}
```

## Reference

<Accordion title="sendCallInvitation" defaultOpen="false">
- **Description**

  Send a call invitation.

- **Function Prototype**

  ```ts
  sendCallInvitation(params: {
      callees: ZegoUser[];
      callType: ZegoInvitationType;
      timeout?: number;
      roomID?: string;      
      data?: string;
      notificationConfig?: ZegoSignalingPluginNotificationConfig;
  }): Promise<{
      errorInvitees: ZegoUser[];
  }>;
  ```

- **Parameter Description**

  | Parameter | Type | Required | Description |
  | :-- | :-- | :-- | :-- | 
  | callees | ZegoUser[] | Yes | List of callee users. A user can invite up to 9 other users at most. |
  | callType | ZegoInvitationType | Yes | Call type. |
  | timeout | number | No | The duration of waiting for the callee response to the invitation, in seconds, with a value range of [1, 600]. The default value is 90. |
  | roomID | string | No | Room ID, generated by you, is a string with a maximum length of 128 bytes. It only supports numbers, English characters, and the following characters: '~', '!', '@', '#', '$', '', '^', '&', '*', '(', ')', '_', '+', '=', '-', ', ';', 'â€™', ',', '.', '&lt;', '&gt;', '/'. |
  | data | string | No | Extension field. |
  | notificationConfig | ZegoSignalingPluginNotificationConfig | No | Offline push configuration. |

</Accordion>
