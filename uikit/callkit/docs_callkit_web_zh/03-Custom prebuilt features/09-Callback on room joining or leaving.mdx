# 监听进出房间回调


## 功能概述

ZEGO UIKit 允许设置部分功能参数，便于开发者在生命周期内触发并监听各种事件。

:::if{props.kitType="LiveStreaming"}
例如，您可能希望当房主（或主播）加入房间时开始计时，并在房主（或主播）离开时结束计时。
:::
:::if{props.kitType="LiveStreaming"}
例如，您可能希望有人加入通话时开始计时，并在其他人都离开离开时结束计时。
:::

## 实现流程

- 如需监听加入房间的回调，请调用 `onJoinRoom` 函数，该函数以将房间中的所有用户（一个数组）作为参数。
- 如需监听离开房间的回调，请调用 `onLeaveRoom` 函数，该函数以房间中的所有用户（一个数组）作为参数。


```javascript
// ....
zp.joinRoom({
    // ...,
    onJoinRoom: () => {
        // 添加您的自定义逻辑
    },
    onLeaveRoom: () => {
       // 添加您的自定义逻辑
    }
    // ...
});
```
