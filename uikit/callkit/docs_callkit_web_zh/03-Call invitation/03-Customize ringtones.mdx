# 自定义铃声

## 介绍

您可以使用 Call Kit 自定义发送和接收通话邀请时的铃声。

<Note title="Note">
本功能仅适用于桌面端浏览器，不适用于移动端浏览器。
</Note>

## 实现流程

```javascript
zp.setCallInvitationConfig({
    ringtoneConfig: {
        incomingCallUrl: 'https://xxx/xxx.mp3', // 接收呼叫邀请时的铃声。
        outgoingCallUrl: 'https://xxx/xxx.mp3'  // 发送呼叫邀请时的铃声。 
    }
})
```
