# 功能概述


**语聊房 UIKit（Live Audio Room Kit）** 提供了一组默认的行为和样式，同时我们支持根据您的需求，灵活地的定制或添加您的自定义业务逻辑。


集成语聊房时，提供 `ZegoUIKitPrebuiltLiveAudioRoomConfig` 用于自定义配置，更多内容可参考 [快速开始](../Quick%20start.mdx)。

目前，我们支持以下自定义功能：
