# Modify user interface text

Live Audio Room Kit (ZegoUIKitPrebuiltLiveAudioRoom)'s UI text provided by the internal components is editable, to modify those, use the `ZegoUIKitPrebuiltLiveAudioRoomInnerText` config.

Here is the reference code:

```dart
class LivePage extends StatelessWidget {
  const LivePage({Key? key, required this.roomID, this.isHost = false})
      : super(key: key);

  final String roomID;
  final bool isHost;

  @override
  Widget build(BuildContext context) {
    return SafeArea(
      child: ZegoUIKitPrebuiltLiveAudioRoom(
        appID: YourSecret.appID,
        appSign: YourSecret.appSign,
        userID: userID,
        userName: 'user_$userID',
        roomID: roomID,

        // Modify your custom configurations here.
        config: isHost
            ? ZegoUIKitPrebuiltLiveAudioRoomConfig.host()
            : ZegoUIKitPrebuiltLiveAudioRoomConfig.audience()
          ..innerText.memberListTitle = 'Members',
      ),
    );
  }
}
```
