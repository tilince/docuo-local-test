## Use a Token
```dart {3,4,7-10}
    ZegoUIKitPrebuiltLiveAudioRoom(
      appID: yourAppID /*input your AppID*/,
      /// appSign: 'do not assign app sign',
      token: $yourToken,
      events: ZegoUIKitPrebuiltLiveAudioRoomEvents(
        room: ZegoLiveAudioRoomRoomEvents(
          onTokenExpired: (remainedSeconds) {
            /// when your token is notified to expired, please renew it.
            ZegoUIKitPrebuiltLiveAudioRoomController().room.renewToken($yourNewToken);
          },
        ),
      ),
      ...
    );
```
